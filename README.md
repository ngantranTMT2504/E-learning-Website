# E-Learning Website

A fullstack web application that allows teachers to create courses and students to register, pay, and learn online.  
Built with **VueJS**, **Spring Boot**, **Hibernate (JPA)**, **SQL Server**, **Bootstrap**, and **VNPay Payment Gateway**.
