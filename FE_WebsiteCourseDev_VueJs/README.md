# library used
1. bootstrap 5.3.7 (npm install bootstrap)
2. axios 1.10.0 (npm install axios)
3. font awesome 3.0.8 (npm i --save @fortawesome/vue-fontawesome@latest-3 @fortawesome/fontawesome-svg-core @fortawesome/free-solid-svg-icons)
4. vuelidate (npm install @vuelidate/core @vuelidate/validators)
5. sweetalert2 (npm install sweetalert2)
6. swiper (npm install swiper)
7. chartjs (npm install chart.js)
8. ps (npm install ps)
